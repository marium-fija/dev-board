Question-1: What are the different ways to select an element in the DOM?

There are many way to select an element in the dom :
1. document.getElementById("id")
2. document.getElementsByTagName("tag")
3. document.querySelector("selector")

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML innerText and textContent is :
innerHTML : sets HTML inside an element.
innerText : sets text inside an element.
textContent : sets all text, including hidden.

Question 3: What is event delegation in the DOM?

Event delegation is a technique where we can attach a single event listener to a parent element and handle events from its children.

Question 4: What is event bubbling in the DOM?

Event bubbling is the process where an event starts from the target element and bubbles up to its ancestors.

Question 5: How do you create, add, and remove elements using JavaScript?

I use this methods to create,add and remove an element in JavaScript :
For create an element : document.createElement()
For add an element : document.getElementById().appendChild()
For remove an element : document.getElementById().removeChild()
